Quick Setup:
1. Create a new Google Sheet
2. Set up 4 sheets: "Games", "Bins", "PlayLog", "Players"
3. In Games sheet: Columns A-E: Game ID, Title, Current Location, BGG ID, Photo URL
4. In Bins sheet: Columns A-B: Bin ID, Description
5. In PlayLog sheet: Columns A-G: Timestamp, Game ID, Players, Notes, Winner, Duration (minutes), Rating
6. In Players sheet: Columns A-C: Name, Email, Color
7. Share the sheet publicly or use API key
8. Copy the Spreadsheet ID from the URL

    var targetSheet = sheet.getSheetByName(data.sheet);
    targetSheet.appendRow(data.row);
  } else if (data.action === 'update') {
    var targetSheet = sheet.getSheetByName(data.sheet);
    targetSheet.getRange(data.row, data.col).setValue(data.value);
  }
